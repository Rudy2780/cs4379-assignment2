## Overview

This assignment demonstrates NumPy fundamentals, pandas data manipulation, and efficient data I/O techniques including CSV/JSON handling and chunked processing.

## Contents

- `notebooks/assignment2.ipynb` - Main assignment notebook with all 16 problems
- `data/` - Generated data files (orders.csv, sensor_data_large.csv, JSON exports)

## How to Run

1. Install required packages: `pip3 install jupyter pandas numpy`
2. Launch Jupyter: `jupyter notebook`
3. Open `notebooks/assignment2.ipynb`
4. Run all cells

## Key Topics Covered

- NumPy array operations, indexing, aggregations, and linear algebra
- Pandas DataFrame manipulation, groupby, pivot tables, and merging
- Efficient CSV reading with usecols and chunking
- JSON I/O with different orientations
- Cross-platform path handling with pathlib
